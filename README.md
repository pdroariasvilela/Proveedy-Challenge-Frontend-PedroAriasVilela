# Landing Page con Vite.js, React, @emotion/styled y React Icons

¡Bienvenido a esta Landing Page!

## Descripción

Esta es una Landing Page desarrollada utilizando varias tecnologías modernas que incluyen Vite.js, React, @emotion/styled, y React Icons.

## Herramientas utilizadas

- **Vite.js**: Vite.js es un nuevo sistema de construcción de aplicaciones web extremadamente rápido y liviano. Utilizamos Vite.js para crear un entorno de desarrollo rápido y optimizado para esta Landing Page.

- **React**: React es una biblioteca de JavaScript para construir interfaces de usuario. En esta Landing Page, hemos utilizado React para crear componentes reutilizables y manejar el estado de la aplicación de manera eficiente.

- **@emotion/styled**: @emotion/styled es una librería de estilos para React que nos permite escribir estilos con sintaxis similar a CSS directamente en nuestros componentes. Hemos utilizado @emotion/styled para estilizar nuestra Landing Page de manera sencilla y mantenible.

- **React Icons**: React Icons es una librería que proporciona iconos populares para usar en aplicaciones React. Hemos incorporado React Icons para añadir iconografía atractiva a nuestra Landing Page.

## Instalación

1. Clona este repositorio en tu máquina local.
2. Ejecuta `npm install` para instalar las dependencias.
3. Ejecuta `npm run dev` para iniciar el servidor de desarrollo y visualizar la Landing Page en tu navegador.
